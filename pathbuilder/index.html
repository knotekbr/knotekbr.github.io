Directions
  • Click in the white space to add vertices
  • Click and drag a vertex to move it
  • The red point (path start/end) can be moved, but not deleted
  • Verticals button draws vertical lines labeled with colors to show the order of stacked lines (R > O > Y > G > B > I > V)
  • Path generates a path (very beta, only reliable for simple polygons), starting and ending at the red point
  • Clear Path removes any generated verticals or path
  • Hide/Show Polygon allows toggling the polygon to easily view the path
  • Reset Tool clears the polygon and any verticals/path
