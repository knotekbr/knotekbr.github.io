Directions
  • Select Move Map or hold the Space key and drag the map to move to the mission location
  • Select Add/Move Vertices and click on the map to build a polygon around the mission area. Existing vertices can be dragged to new locations
  • Select Delete Vertices and click an existing vertex to remove it
  • Build Path will generate the path that covers the polygon
    • The path is displayed as red lines
    • A blue circle traces the path to show waypoint order
    • Regions generated during this process are displayed as multicolored backgrounds
  • Clear Path removes the existing path, region colors, and trace animation
  • Hide/Show Polygon toggles polygon visibility to make the path easier to view
  • Reset Tool resets the tool to its initial state
