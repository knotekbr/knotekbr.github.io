Build Path Tools Clear Path Hide Polygon Reset Tool
Directions
  • Click in the white space to add vertices
  • Click and drag a vertex to move it
  • The red point (path start/end) currently has no effect
  • Build Path will generate the path that covers the polygon
    • The path is displayed as red lines
    • A blue circle traces the path to show waypoint order
    • Regions generated during this process are displayed as multicolored backgrounds
  • Clear Path removes the existing path, region, and trace animation
  • Hide/Show Polygon toggles polygon visibility to make the path easier to view
  • Reset Tool clears the polygon, path, regions, and trace animation
